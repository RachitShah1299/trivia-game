# Trivia Quiz App

Welcome to the Trivia Quiz App! This is a simple web application that allows users to take a trivia quiz based on their chosen category and difficulty level.

## Features

- Choose a quiz category and difficulty level.
- Answer multiple-choice questions in the quiz.
- See your current test score and overall test score.
- Navigate through quiz questions and see the result at the end.

## Getting Started

### Prerequisites

Before you begin, make sure you have the following installed:

- Node.js
- npm (Node Package Manager)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/RachitShah1299/trivia-game.git
cd trivia-quiz-app
```

2. Install the required dependencies:

```bash
npm install
```

## Usage

1. Start the development server:

```bash
npm start
```

2. Open your web browser and navigate to `http://localhost:3000` to use the app.

3. Choose a quiz category and difficulty level, and start the quiz. Answer the questions to see your scores.

## Dependencies

This project uses the following major dependencies:

- React
- React Router DOM
- Axios
- Zustand (for state management)
- Tailwind CSS (with Tailwind Merge)
- TypeScript

For a complete list of dependencies, refer to the `package.json` file.

- `npm start`: Start the development server.
- `npm build`: Build the production version of the app.
- `npm test`: Run tests using Jest.
